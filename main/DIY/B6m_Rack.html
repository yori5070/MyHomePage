小Ｂ６版用・ラック

　娘の大量のコミックを収める頑強なラックを作る。

　小B6サイズ（112mmｘ174mm）のコミック用のラックで、コンパネ（12mm厚）でホゾ組をして接着。積み上げることを想定しています。

　写真のように本が触れる内側だけを塗装、木口はクリアラッカー仕上げまでで、この後色をつけて下地を隠す予定です。

　コンパネ２枚と背板用の5.5mm合板１枚で８つ作ることができます。ひとつのラックに３０冊前後収容できます。

タンポ擦り仕上げ

　合版のラックの内側になる面は、クリアラッカーでタンポ擦り仕上げとした。工程は以下のとおり。

　パテで傷を埋める（ポリパテ推奨、木工パテは研磨に難あり） → 120番で木地調整 → 砥の粉で目止め → ウッドシーラー下塗り → 180番でサンディングしケバをとる → サンディングシーラー中塗り（２回） → 320番でサンディング → クリアラッカー上塗り（２回） → 600番で水研ぎ（サンディング） → クリアラッカーでタンポ擦り仕上げ（５回〜７回） という工程で行った。
